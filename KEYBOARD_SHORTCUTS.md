# LazyCat Terminal 快捷键

## 标签页管理
- `Ctrl + Shift + T`: 新建标签页
- `Ctrl + Shift + W`: 关闭当前标签页 *
- `Ctrl + Tab`: 切换到下一个标签页
- `Ctrl + Shift + Tab`: 切换到上一个标签页
- `Ctrl + PageUp`: 切换到上一个标签页
- `Ctrl + PageDown`: 切换到下一个标签页

## 终端分屏
- `Ctrl + Shift + J`: 垂直分屏（左右）
- `Ctrl + Shift + H`: 水平分屏（上下）
- `Alt + h`: 选择左侧终端
- `Alt + l`: 选择右侧终端
- `Alt + k`: 选择上方终端
- `Alt + j`: 选择下方终端

## 终端窗口管理
- `Ctrl + Shift + Q`: 关闭当前分屏的VTE终端 *
- `Ctrl + Alt + Q`: 关闭当前标签页中除当前VTE外的所有其他VTE终端 *

**注意**：标记 * 的操作会自动检测VTE中是否有正在运行的进程。如果检测到活跃进程，会弹出确认对话框，只有用户确认后才会杀掉进程并关闭VTE。窗口关闭时也会进行相同的检测。

## 剪贴板操作
- `Ctrl + Shift + C`: 复制
- `Ctrl + Shift + V`: 粘贴
- `Ctrl + Shift + A`: 全选

## 字体大小调整
- `Ctrl + +`: 增大字体
- `Ctrl + -`: 减小字体
- `Ctrl + 0`: 重置字体大小

## 透明度调整
- `Ctrl + 鼠标滚轮`: 调整窗口透明度

## 后台任务完成提示

当你在后台标签页中执行命令后切换到其他标签时，如果后台任务完成（terminal title 发生变化），该标签的文字会自动变为**金黄色**，提示你任务已完成。

- **触发条件**：在标签中按下任何键 + 标签在后台 + terminal title 变化
- **高亮颜色**：金黄色 `#FFD700`
- **取消高亮**：切换到该标签时自动清除高亮

**使用示例**：
1. 在标签 1 中运行 `make -j8` 编译项目
2. 切换到标签 2 继续工作
3. 编译完成时，标签 1 的文字自动变为金黄色
4. 切换回标签 1 查看结果，高亮自动消失

## 进程检测功能

在关闭 VTE 终端时，系统会自动检测是否有正在运行的进程（除了shell本身）：

- 如果检测到活跃进程：弹出确认对话框，只有用户确认后才会杀掉进程并关闭终端
- 如果没有活跃进程：直接关闭终端

确认对话框特性：
- 无边框透明设计，与终端风格一致
- 对话框边框使用 VTE 前景色（默认绿色）
- 使用与主窗口相同的投影效果
- 右上角关闭按钮使用 VTE 前景色（默认绿色），hover/click 时透明度变化
- 可以通过点击关闭按钮或按 Esc 键取消关闭操作
- 警告文本和确认按钮使用与终端相同的前景色（默认绿色）
- 确认按钮默认状态完全透明背景，只显示边框（移除 GTK 默认 focus 效果）
- hover/click 时按钮显示半透明背景
- 对话框打开时自动聚焦到"确认结束"按钮
- **可以直接按 Enter 键快速确认操作**
